# Todo App

A simple, responsive todo web application built with vanilla HTML, CSS, and JavaScript.

## Features

- **Add tasks** — type in the input and press Enter or click Add
- **Complete tasks** — check the checkbox to mark a task done (strikethrough)
- **Edit tasks** — click on a task's text to edit inline, confirm with Enter or click away
- **Delete tasks** — click the × button to remove a task

## Usage

Open `index.html` in any modern browser — no build step or server required.
